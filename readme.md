## Technologies Used
* `Backend: Node/Express`
* `MongoDb`
* `Libraries: Express`

## To Install
* `Download or clone`
* `Open the terminal and cd into the cloned or downloaded folder from the terminal`
* `Type npm install to install all dependencies`
* `check sample.env and update all environmental variable`
